PROJECTS

GESTURE RECOGNIZED AI CONTROL ENVIRONMENT (GRACE)

A gesture-recognition prototype built in Matter IoT domain, with the present use-case of home-automation, which is useful to control the IoT appliances of a room just by the gestures made by hand. This working prototype uses Infineon's XENSIV 60GHz Radar Sensor and with Radar Fusion GUI it can tell us the gestures made and also will do the specified task.

AUTOMATED OBJECTIVE-TYPE QUESTION GENERATION

Published in TIJER Journal

An Edtech project wherein a machine learning model automates the generation of objective-type questions, some of which including MCQs, Boolean type, Assertion and Reasoning etc, from given input text. It uses the OpenAI`s ChatCompletion Model and Python modules to parse input, generate questions, organize them into a CSV file, incorporating metrics like Difficulty Level and Clarity Score. The tool enhances educational content creation and can be integrated into e-learning platforms.

DROP IT (AI TRASHBIN)

DROP IT is an AI-based waste disposal system that tackles waste management issues by automating waste segregation and rewarding the public upon proper trash disposal. Uses FOMO and Edge-Impulse ML Platform for Model Training. Hardware components like Arduino, ESP32-Cam, Servo motors, Ultrasonic sensors, IR Sensors, etc., are used. All the IoT devices are integrated with the ThingsBoard Cloud Platform for real-time monitoring and predictive analytics.

MEDIA COLORIZER

Colorizer is an AI-powered tool that uses advanced deep learning and CNN technology to transform grayscale images and videos into vibrant colors. It is ideal for film restoration, historical media enhancement, and artistic projects, ensuring seamless processing and color consistency across frames.
